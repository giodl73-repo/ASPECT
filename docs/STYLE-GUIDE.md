# DEGAS Style Guide

## Writing in a Persona's Voice

Each persona in `personas/` has a documented voice. When writing panel reviews, stay inside that voice. Key principles:

- **Specificity**: personas name specific elements of the design, not vague impressions. Tufte says "this gridline" not "the clutter." Bertin says "position encodes income but income is quantitative — you have used hue, which is not an ordered variable" not "the encoding is wrong."
- **Historical grounding**: personas reference their own work and their contemporaries' work as evidence. Minard cites the Russian campaign map. Neurath cites the Vienna Museum's ISOTYPE output.
- **Productive disagreement**: when personas disagree, the disagreement is substantive, not procedural. Tufte and Kandinsky genuinely see the same work differently because they have different theories of what visualization is for.
- **No consensus framing**: the SUMMARY does not say "the panel agrees." The SUMMARY names the tensions and lets the designer decide.

## Writing Design Specifications

- Describe visuals precisely enough that a trained practitioner could render them
- State all encodings as rules: variable = data element
- Name school-specific techniques by their school vocabulary
- Include specific color values where possible (hex or named CIELAB values)
- For any constraint that is unusual or school-specific, explain why

## Writing Briefs

- The core message should be a single sentence: the viewer should leave with _X_
- Audience description should be specific enough to guide design: not "general public" but "adult readers of a national newspaper, assumed to have seen a bar chart before, unlikely to have seen a flow map"
- School assignment can be provisional — note uncertainty if present

## About the Schools

No school is superior to another. Statistical graphics is not more rigorous than abstract art. Isotype is not simpler than cartography. Each school has its own demands and its own criteria for success. A work succeeds or fails within its school, not against some universal standard.

When a work mixes schools, name the mix. "This work operates primarily in the statistical graphics school but deploys ISOTYPE's repetition principle for one categorical layer." That is a legitimate design choice; it is not allowed to be invisible.
