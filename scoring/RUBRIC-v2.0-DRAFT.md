# DEGAS Rubric v2.0

**Forward-only. Amendments ratified from clustered innovations (2+ across 2+ works).**  
**Advisory threshold: < 60. Binding threshold: 60+.**

*This version consolidates 17 amendments from v1.0–v1.12 into a cleaner structure. All innovations are preserved; redundant steps are merged; every dimension follows the same three-layer pattern: Classification → Anchors → Modifiers.*

---

## Structure (100 points total)

| Dimension | Pts | Core question |
|-----------|-----|--------------|
| 1. School Fidelity | 20 | Does the work speak the grammar of its claimed school? |
| 2. Integrity | 20 | Does the work honestly represent what it claims to show? |
| 3. Legibility | 15 | Can the intended audience, in the intended context, read it? |
| 4. Execution | 15 | Does every mark earn its place, and is it well-made? |
| 5. Resonance | 15 | Does the work create the intended effect? |
| 6. Purpose | 15 | Does the form serve the function? Does the work take its subject seriously? |

## 1. School Fidelity (20 pts)

### Classification — what is this work's relationship to its school?

| Type | Definition |
|------|-----------|
| **A — Member** | Works within an established school; uses its conventions deliberately |
| **B — Founder** | Predates or establishes the associated school; grammar must be derivable (declared grammars reach the upper anchors more readily — integrate this into scoring, not a separate step) |
| **C — Synthesis** | Deliberately operates in two or more schools simultaneously; synthesis declared or clearly intentional |
| **D — Accidental mix** | Mixes school vocabularies without coherent resolution |

**School-story match** — before scoring, ask: is this the right school for this story? A work can apply its school's grammar perfectly and still have chosen the wrong school.

| Match | Effect |
|-------|--------|
| Well-matched | None |
| Partial mismatch (another school would serve better) | Note in review; −0 to −2 |
| Significant mismatch (grammar obscures the story) | −2 to −4; recommend alternate school |

### Anchors

*Type A (Member):*

| Score | Anchor |
|-------|--------|
| 18–20 | The grammar is fully and correctly applied. An expert in the school recognizes it immediately. Convention breaks are named and justified. |
| 13–17 | Strong fidelity; minor lapses. Core grammar respected. |
| 8–12 | Partial fidelity; significant imports from other schools, or superficial application. |
| 3–7 | Nominal claim; surface resemblance only; grammar absent or violated. |
| 0–2 | No demonstrable fidelity; incompatible with the claimed school. |

*Type B (Founder):* Score on whether the founding grammar is coherent, learnable, and repeatable. Works that declared their grammar at founding (Playfair's preface, Kandinsky's diagrams-as-demonstration) reach the 18–20 band more readily than those whose grammar must be reconstructed.

| Score | Anchor |
|-------|--------|
| 18–20 | Founding grammar is coherent, complete, and derivable from this work alone. Declared grammars reach this band. |
| 13–17 | Strong founding principles with gaps; or derivable but requiring expert reconstruction. |
| 8–12 | Some principles present but embedded in idiosyncrasy. |
| 0–7 | Isolated techniques; no recoverable system. |

*Type C (Synthesis):* Average fidelity to each participating school on Type A anchors. Add up to +2 if the synthesis is named and tensions resolved.

*Type D (Accidental mix):* Score against the most-violated school's grammar. Cap at 12.

## 2. Integrity (20 pts)

### Pathway — is this work reportive or demonstrative?

**Before applying any sub-questions, determine the encoding mode:**

| Mode | Definition | Apply |
|------|-----------|-------|
| **Reportive** | The work represents external data; accuracy assessed by correspondence between marks and data | Sub-questions 1–3 below |
| **Demonstrative** | The diagram proves a claim by construction; viewer verifies by following the argument | Sub-questions E1–E4 below |

*Examples: Snow's cholera map = reportive. Euclid's geometry diagrams = demonstrative. Kandinsky's compositional diagrams = demonstrative.*

---

**For reportive works — three questions:**

**Question 1 — Disclosure:** Does the work declare what it is claiming to show?

| Situation | Effect |
|-----------|--------|
| Source data quality declared (uncertainty, provenance, limitations) | None |
| Key variables encoded; exclusions noted where they create framing effects | None |
| Confidence suppressed (known limitations undeclared) | −2 to −4 |
| Variable exclusion creates active misleading (title claims X; X not encoded) | −2 to −3 |
| Categories embed moral/political judgment as if empirical fact (ontological contamination) | −3 to −8; declared contamination reduces dock by half |

**Question 2 — Argument integrity:** Does the work make only what the data can support?

| Situation | Effect |
|-----------|--------|
| All claims supported by the data; accuracy tradeoffs declared | None |
| Causation implied by proximity/structure; plausible but not evidenced | −1 to −2 |
| Accuracy tradeoff (less precise form chosen) declared and proportionate | −0 to −1 |
| Accuracy tradeoff undeclared; causal claim overstated | −2 to −4 |
| Claim contradicted by the data | −5 to −6 |

**Question 3 — Framing integrity:** Is the conceptual model honestly presented?

| Situation | Effect |
|-----------|--------|
| Standard data model; no novel analytical framework | None |
| Novel framework declared in-work (diagrams demonstrate, preface explains) | −0 to −1 |
| Novel framework in adjacent documentation | −1 to −2 |
| Novel framework undeclared but inferable | −2 to −3 |
| Novel framework undeclared and non-obvious; viewer must supply the founding premise | −3 |

---

**For demonstrative works — four questions (E1–E4):**

| Question | Ask | Effect if failed |
|----------|-----|-----------------|
| E1 — Foundation | Are the premises the argument builds from stated or established? | −1 to −2 per undeclared premise |
| E2 — Completeness | Are all steps visible? | −1 to −3 per missing step |
| E3 — Validity | Does the construction actually prove what it claims? | −3 to −5 for logical gaps |
| E4 — Convention | Are the diagram's operating rules stated or conventionally established? | −1 to −2 |

---

**Main anchors (apply after questions 1–3 or E1–E4):**

| Score | Anchor |
|-------|--------|
| 18–20 | The work is fully honest about what it claims, how it argues, and what it assumes. Disclosures complete. No unjustified claims. |
| 13–17 | High integrity with minor issues. Modest undisclosed uncertainty or mild causal implication. Nothing that materially misleads. |
| 8–12 | Detectable issues: significant suppression, moderate overreach, undeclared novel framework, or moderate category contamination. |
| 3–7 | Significant distortions that would mislead a typical viewer. |
| 0–2 | The work is materially false, deceptive, or built on categories that invert the underlying reality. |

## 3. Legibility (15 pts)

### Context specification — before scoring, state:

| Field | Specify |
|-------|---------|
| **Primary audience** | Who the work was designed for |
| **Reading context** | *Wayfinding* (task under time pressure) / *political presentation* (accompanied) / *contemplative study* (solo, extended) / *public display* (glanced, passing) |
| **Narration mode** | *Standalone* (no accompaniment expected) / *accompanied* (narration is part of the encoding — score the narrated experience; note standalone legibility in brackets) |
| **Contemporary context** | How the work is now primarily encountered, if different; note decay in brackets [contemporary: X/15] |

### Anchors (applied to primary context)

| Score | Anchor |
|-------|--------|
| 13–15 | The intended audience, in the primary context, extracts the necessary information without friction. The design serves that context's demands precisely. |
| 9–12 | Good legibility with minor friction in the primary context. Core message accessible. |
| 5–8 | Partial legibility. The audience can understand with effort or guidance beyond what the context provides. |
| 2–4 | Significant failure in the primary context. |
| 0–1 | Inaccessible. |

### Modifiers

- **Living artifacts**: works continuously modified after the founder's death. Specify `version_reviewed` and note institutional divergence.
- **Dynamic works**: for time-animation or interactive works, score the dynamic experience as primary; append [static: X/15].

## 4. Execution (15 pts)

*Replaces Visual Economy + Craft as two lenses within one dimension.*

### Economy lens — does every mark earn its place?

Before scoring, classify visual complexity:

| Type | Treatment |
|------|-----------|
| **Data marks** | Required; do not penalize |
| **Intrinsic complexity** | Subject requires high density (biological morphology, ecological systems, dense networks, ecology-as-composition) — score only excess above intrinsic necessity |
| **Structural-attentional** | Serve display function without encoding data — light penalty (−1 to −2 total) |
| **Cognitive-accessibility marks** | In low-literacy / public-display contexts: do not penalize if they demonstrably reduce cognitive steps to comprehension |
| **Decorative marks** | Penalize — these are chartjunk |

### Craft lens — are the necessary marks well-made?

Assess: typography, color control, consistency of similar elements, alignment, print/rendering quality relative to period constraints.

*Period-constraint note:* distinguish limitations imposed by production technology (historical works) from discretionary design choices. Period-constrained limitations should be noted but penalized less severely than discretionary failures.

### Anchors (both lenses combined)

| Score | Anchor |
|-------|--------|
| 13–15 | Every mark earns its place by data, intrinsic necessity, structural function, or cognitive accessibility. Execution is precise, consistent, and polished. |
| 9–12 | High economy and good craft with minor issues — one excess element, one minor inconsistency. Nothing that significantly burdens the reading. |
| 5–8 | Detectable excess or craft failures that undermine trust or compete with the data. |
| 2–4 | Significant noise or careless execution. The data fights its surroundings. |
| 0–1 | The design is dominated by unjustified elements or is technically incompetent. |

## 5. Resonance (15 pts)

### Classification — what type of resonance does this work primarily achieve?

| Type | Definition | Score on |
|------|-----------|---------|
| **Experiential** | The work creates a direct experience — aesthetic, emotional, or perceptual — through design choices (Type I), encoding fidelity (Type II), or immediate visual shape (Type IV). These three mechanisms all produce the same kind of effect in the viewer. | Whether the experience is fully achieved; whether design choices consistently serve it |
| **Advocacy** | The work is designed to move a specific audience toward action | Whether the audience is correctly identified, the form appropriate, and the goal achieved or achievably aimed at |
| **Domain** | Full resonance available immediately to domain-initiated viewers (practitioners, regular users); partial or absent to others | Whether the domain-initiated viewer receives immediate, complete recognition |

*Timing note:* for works whose resonance was delayed (achieved strongest impact posthumously), note the delay mechanism in the SUMMARY (epistemic, political-constructed, or institutionally transmitted) and apply +0 to +1 bonus for politically-constructed delay where the delay itself demonstrates the work's threat to established power. This replaces the Type VI mechanism matrix.

*Dynamic works:* for works with animation or interaction as primary variable, score the dynamic experience; append [static: X/15].

### Anchors (applied to primary type)

| Score | Anchor |
|-------|--------|
| 13–15 | The work fully achieves its primary resonance type. The effect is clear, consistent, and persists after the reading. |
| 9–12 | Solid resonance with minor misfires. The emotional register / advocacy impact / domain recognition is right but slightly muted. |
| 5–8 | Partial resonance. The work communicates correctly but the primary effect fails or only partially fires. |
| 2–4 | Wrong effect: confusion where there should be affect, inertia where there should be urgency. |
| 0–1 | No detectable resonance. |

### Modifier
Each additional resonance type that is also achieved contributes +0 to +1 bonus (maximum 15 total).

## 6. Purpose (15 pts)

*New dimension. Consolidates: school-story match overflow, moral weight proportionality, and the question of whether design serves function.*

Does this work take its subject and audience seriously? Is the form chosen appropriate for the purpose? Does the work do what it set out to do — not just technically, but intentionally?

### Classification — what is this work for?

Name the primary purpose before scoring:
- **Functional** (navigation, reference, decision support): score on whether the design serves the task efficiently
- **Informational** (understanding, learning, exploring): score on whether the design supports comprehension
- **Advocacy** (persuasion, mobilization, policy change): score on whether the design serves the argument's stakes
- **Aesthetic** (experience, beauty, wonder): score on whether the design achieves the intended aesthetic effect
- **Demonstrative** (proof, teaching, argument by construction): score on whether the design serves the logical structure

### Anchors

| Score | Anchor |
|-------|--------|
| 13–15 | The form fully serves the purpose. For works on subjects with clear human stakes (preventable deaths, injustice, climate impact), the resonance is proportionate to those stakes — the design does not treat urgent subjects as if they were merely interesting. The work does what it set out to do. |
| 9–12 | The form mostly serves the purpose. Minor misalignments — a slightly wrong tone, a form that underserves the urgency, or one element that pulls against the purpose. |
| 5–8 | Partial purpose-form alignment. The work is correct in its encoding but fails at its stated function. An inert chart on a subject that deserved urgency; a functional tool that treats wayfinding as aesthetic problem. |
| 2–4 | The form works against the purpose. A beautiful display of data that needed to be a tool. An advocacy work so visually complex it cannot reach its audience. |
| 0–1 | The form and purpose are incompatible. The work has failed at the level of intention. |

### Modifiers
- **School-story match** (from School Fidelity Step 0c): a significant school-story mismatch is also a Purpose failure. The penalty can be split between the two dimensions at the reviewer's discretion.
- **Moral weight proportionality**: for works on subjects with clear human stakes, the panel should explicitly assess whether the resonance is proportionate. This is scored here, not in Resonance.

- **Disclosure, argument, framing** integrity (consolidates v1.2 A/B/C + v1.4 D + v1.7 E + v1.10 F + v1.9 L)

- **School-story match** (v1.9 K) → retained in School Fidelity + overflow to Purpose

- **Ontological categories** (v1.10 N) → Question 1 in Integrity

- **Moral weight proportionality** → Purpose dimension

## What Changed

- **Visual Economy + Craft merged** → Execution (15 pts): these two dimensions were measuring the same thing at different resolution; merging them restores compositional balance and reduces reviewer decision fatigue
- **Purpose dimension added** (15 pts): school-story match, moral weight proportionality, and function-form alignment finally have a home; previously scattered across School Fidelity and Resonance
- **Resonance I+II+IV collapsed** to Experiential: these three types all ask "does the work create an experience?" — the mechanism differs but the criterion does not; reviewers were applying the same anchors regardless of which type they declared
- **Type VI mechanism matrix removed**: speed × agency produced +0 to +1 bonus with high analytical overhead; replaced by a simpler timing note that the SUMMARY handles in prose
- **J modifier absorbed**: adds at most 1 point; the Type B anchors now handle this in text
- **Dimension structure standardized**: every dimension now follows Classification → Anchors → Modifiers
