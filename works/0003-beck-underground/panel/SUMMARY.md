## Key Agreements

**Topology is not distortion.** All three personas accept Beck's data model argument: the network is the data; geographic position is not. The map correctly encodes network connections. This resolves the v1.1-era worry that geographic distortion would be a systematic Data/Story Integrity failure for the schematic-cartography school. It is not — provided the designer's data model is stated. Beck states it clearly in his submission memo and in the design itself.

**Visual Economy is excellent.** All three personas give 14 or 15. The map has nothing removable. Beck's own analysis of what he removed (street grid, terrain, landmarks, north arrow, scale bar, decorative border) is exhaustive and correct. Tufte notes the data-to-ink ratio approvingly; Bertin notes the correct assignment of visual variables; Beck notes the functional completeness.

**Resonance is limited.** All three personas score 12 or 13. This is a tool, not an art object. Its resonance is functional (task-completion satisfaction for passengers) or domain-gated (network gestalt for those who know the system). Neither of these is one of the current four Resonance types. The low convergent score from three different reviewers is itself an innovation signal.

---

## Key Tensions

**Beck vs. Tufte on geographic context.**  
Beck scores Data/Story Integrity 19; Tufte scores 16. The gap is on the undeclared variable scale. Beck holds that the central expansion is not a data integrity issue because spatial proportion is not in his data model. Tufte holds that even if spatial proportion is not in the data model, the viewer assumes it, and the unannounced violation of that assumption is a mild integrity issue (Step B: undeclared tradeoff). Both positions are defensible; the rubric's Step B sub-step is doing the work of locating the disagreement precisely.

**Beck vs. Tufte/Bertin on Legibility.**  
Beck gives 15; Tufte 13; Bertin 14; Audience 12. Beck insists on scoring against the original design audience in the original use context. Tufte and Audience insist on scoring against the contemporary full user population. This is the sharpest methodological disagreement so far. The rubric does not resolve it — it forces a single score. The Audience lens's three-audience breakdown (14, 10, 9) is the most useful analysis, but it cannot produce a single number without a weighting decision.

## Cluster Updates

### Cluster D — Legibility context-dependence: **AMENDMENT THRESHOLD MET**
**Works: 0002 + 0003. Innovations: #17, #20 (0002) + #29, #30 (0003)**  
The 1933 Underground map is the definitive test case: its Legibility score ranges from 14/15 (original use context) to 9/15 (contextless contemporary reading), a spread of 5 points within a single work. Innovation #29 (context-decay) directly confirms #17 (context-dependent legibility) from Nightingale. Innovation #30 (living artifact) directly confirms #20 (campaign artifact) from Nightingale. Threshold met across two works. *Amend Legibility.*

### New cluster forming — Resonance Type V (domain-gated / functional): **WATCH**
**Work: 0003. Innovations: #25 (domain-gated gestalt), #31 (functional resonance)**  
Two innovations in one work, both pointing at the same gap: Resonance Types I–IV do not capture forms of resonance that are either audience-gated (only the initiated can access the network gestalt) or task-based (the satisfaction of a tool that works). These are meaningfully different from aesthetic, data-emergent, advocacy, and visual-gestalt resonance. *Watch for confirmation in work 0004.*

### New cluster forming — Data/Story Integrity and data model declaration: **WATCH**
**Work: 0003. Innovations: #22 (topology as data model), #24 (undeclared variable scale)**  
Both point at the same gap: the Data/Story Integrity sub-steps currently address source quality, accuracy tradeoffs, and causation — but not the designer's *data model*. A designer who works with a non-standard data model (network topology rather than geographic space) can encode correctly *relative to their model* while appearing to distort *relative to a different model*. The rubric should ask "what model?" before asking "is the encoding faithful?" Watch for work 0004.

## Design Recommendations

*(Canonical import — rubric and pipeline recommendations only.)*

1. **Amend Legibility (Cluster D confirmed).** Add multi-context scoring: specify primary scored audience and context; note secondary contexts with their scores. For canonical imports, score against original design context primarily; note contemporary context decay.

2. **Add `version_reviewed` to canonical import frontmatter.** Living artifacts (works modified by institutions after the founder's death) require specifying which version is assessed and how it diverges from the contemporary institutional version.

3. **Watch for Resonance Type V.** Domain-gated gestalt and functional resonance are both candidates. Need confirmation across one more work before amending.

4. **Watch for data model declaration sub-step.** Topology-as-data-model vs. geography-as-data-model: the rubric should ask "what does the designer claim to be representing?" before assessing faithfulness. Need one more confirmation.

5. **Document schematic-cartography school** formally in `schools/` using this work's design description as the grammar source. The 1933 map is the founding document; the grammar in the design description is extractable and stable.
