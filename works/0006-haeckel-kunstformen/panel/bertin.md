## Review

**Visual variable audit:**

The task of auditing Haeckel's plates in terms of *Sémiologie Graphique* is unusual because the primary encoding is neither symbolic nor abstract — it is iconic. Each organism is depicted as itself, with visual variables working simultaneously at two levels: the semiotic level (what the variable encodes about the data) and the perceptual level (how the variable is decoded by the viewer's visual system).

*Species identity → morphological form:* Position, shape, texture, color, and value all cooperate to encode species-specific morphological features. This is what I call a **compound encoding** — no single variable carries the information; the organism's identity emerges from the combination. This is neither correct nor incorrect by the usual rubric; it is a different grammar. The biological illustration school has its own variable system.

*Taxonomic relationship → plate grouping:* One plate, one taxonomic group. This is a categorical encoding using spatial segregation — the plate boundary acts as a container defining group membership. The encoding is correct: shape (species identity) within the container communicates "same group." Color would be redundant to distinguish in-group variation; the plate-boundary encoding is efficient.

*Scale → unstated, legend required:* This is the work's most significant variable-matching failure. Scale is a quantitative variable — radiolaria range from 0.1 to 2 mm; the plates render them at 1–4 cm diameter. The implied scale of representation varies across plates with no visual declaration. The variable (implied physical size of depicted organisms) is not matched to any encoding on the plate itself. A viewer studying comparative morphology across plates cannot make size comparisons without consulting separate text. Score: variable-matching failure, -2 on Data/Story Integrity.

*Relative size within a plate:* Variable across specimens without declaration. When Haeckel arrays radiolarian skeletons of different sizes on the same plate, the viewer cannot distinguish "this specimen is larger" from "Haeckel chose to render it larger." The size variable is not encoding data.

**On ornamental borders and decorative framing:**

The non-biological decorative elements — borders, ornamental framing, the Art Nouveau geometric borders on some plates — are marks that encode nothing about the organisms depicted. By my metric, they are chartjunk in the strict sense: ink that is not data. However, I must note that for a work at this scale of craft execution (see Craft score), the borders serve a compositional function: they define the plate as a unified composition, separate it from the page, and direct the eye inward. This is what I would call *structural support* rather than data encoding or pure decoration. It is not chartjunk. It is closer to the axis line in a chart: not data, but not removable without cost. Score: Visual Economy 9/15 — significant non-data ink, but most of it is structural rather than gratuitous.

**On the founding grammar (Type B assessment):**

The principles recoverable from Haeckel's plates — accuracy-as-foundation, composition-for-diagnostic-display, population-at-scale, color-as-morphological-information — form a learnable grammar. A subsequent illustrator studying these plates would know the rules. The gap: scale declaration. A practitioner who learned from Haeckel alone would not know how to handle cross-plate size comparison, because Haeckel did not solve it. Score: 16 (strong founding grammar with one recoverable gap).

## Innovations Flagged

1. **Compound iconic encoding** — when biological accuracy requires that multiple visual variables cooperate simultaneously to encode organism identity, no single variable analysis (in the Bertinian sense) is sufficient. The variables are not separable. This suggests the rubric needs a note for illustration-school works: compound iconic encoding should be assessed as a system, not variable-by-variable. *Rubric anchor: School Fidelity / Data/Story Integrity.*

2. **Scale ambiguity as systematic omission** — when a scientific illustration school uses variable rendering scales without visual declaration, relative-size comparison across plates is disabled. This is a structural deficit of the school's founding grammar, not an individual plate's failure. *Rubric anchor: Data/Story Integrity (Step A/D).*
