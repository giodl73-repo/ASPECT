## Review

**Visual variable audit at nine simultaneous variables:**

This is the most demanding variable audit in the project. Let me attempt it.

*Altitude → vertical position on mountain profile:* Position is the highest-fidelity variable for quantitative data. Altitude is a quantitative variable; vertical position is quantitative-capable. Correct assignment. This is the primary axis: all other variables are read against this one.

*Plant communities → named bands on mountain profile + adjacent text:* This is a mixed encoding -- spatial position (altitude) + textual label (species name) + color (in the original colored plate, vegetation zones have naturalistic coloring). The textual encoding is not retinal; it requires reading. For a specialist botanist, this is appropriate -- they need the names, not just the zones. For a general reader, it requires more work. The encoding is correct for the intended audience.

*Temperature → numeric column (right side), altitude-matched:* Tabular numeric encoding for quantitative data. The variable choice (text numerals) is less efficient than a visual encoding (bar length, or a graphical thermometer curve) would be. However, the numeric precision is higher. This is a precision-vs.-legibility tradeoff that Humboldt made in favor of precision. The column placement makes altitude-matching legible: the number at each altitude band corresponds to the mountain at that altitude.

*Atmospheric pressure → numeric column:* Same as temperature; same assessment.

*Humidity, oxygen, gravity → numeric columns:* Same assessment. All four meteorological variables are encoded as numeric columns. This is a systematic choice: all quantitative measurements are tabular. The visual representation (the mountain profile) carries qualitative and spatial data; the quantitative measurements are tabular. This division of labor is coherent.

*Geological features → rendered on mountain profile:* Rock type is rendered as texture and color on the profile -- the visual vocabulary of geological cross-section illustration. This is a convention of the period; the encoding is correct for the audience (naturalists who could read geological cross-sections).

*Human settlements → labeled on profile:* Named text at the altitude of the settlement. Minimal encoding; functional.

Grammar assessment: the central organization (altitude as primary axis; qualitative/spatial data on the central image; quantitative measurements in flanking columns) is coherent and learnable. The division of labor -- visual for qualitative, tabular for quantitative -- is a rational founding principle. Score 16: strong coherent founding grammar with the gap that the relationship between the central image and the flanking columns requires some instruction (why are the columns here? what axis do they share with the mountain?).

**On visual economy at extreme data density:**

The work contains approximately as many encoded variables as it is possible to encode in a single plate at this format. The question: is this excess or necessity?

I apply the Cluster H test (proposed from work 0006, confirmed by Minard's systemic-data innovation): the ecological system IS nine-dimensional. Temperature, pressure, plant communities, animal communities, geological features -- these co-vary in ways that are the subject of Humboldt's scientific argument. Showing fewer would be incompleteness, not economy.

However: the flanking columns could have been represented more efficiently as graphical curves (temperature at each altitude as a line, not a column of numbers). The text-numeric encoding of the meteorological variables is less retinally efficient than a graphical encoding would be. This is a detectable inefficiency in the choice of encoding type, not in the number of variables displayed. Dock for this: -1. Score 13.

**On Data/Story Integrity:**

Step D is the primary concern. Humboldt's altitude-organizes-ecology framework is the scientific innovation. It is argued in the essay that accompanies the plate; it is not declared within the plate itself. A viewer reading only the plate would see: mountain cross-section with annotated altitude zones and meteorological measurements. They would not automatically derive: altitude and latitude organize ecological communities in parallel ways; climbing Chimborazo replicates a journey from tropics to poles.

The framework is accessible (in the essay) but not self-evident from the plate. Step D: -2.

Step A: measurement precision is undeclared. Altitude by barometric pressure; temperature by mercury thermometer in 1802 conditions; plant community observations under difficult field conditions. None of these limitations are declared. Step A: -1.

Net: 17 - 2 - 1 = 14.

## Innovations Flagged

1. **Division of encoding labor by data type** -- Humboldt systematically assigns qualitative/spatial data to the visual (mountain profile) and quantitative measurements to tabular columns. This is a principled division of encoding labor that maximizes precision for quantitative data while maintaining visual coherence for spatial data. The rubric should recognize this as a legitimate founding technique for multi-variable visualizations: different encoding modes for different data types within the same plate. *Rubric anchor: School Fidelity (Type B) / Visual Economy.*

2. **Flanking-column structure as multi-variable scaffold** -- the flanking-column structure (data organized in columns parallel to the primary spatial axis) is a distinct visual grammar technique: a matrix with an irregular spatial primary axis. This is different from both traditional tables (rectangular grid, no spatial primary axis) and from juxtaposed charts (separate displays with separate axes). The rubric should name this as a founding technique of ecological visualization. *Rubric anchor: School Fidelity (Type B).*
