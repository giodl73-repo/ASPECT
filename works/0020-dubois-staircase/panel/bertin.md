# Panel Review — Bertin on 0020: Du Bois Staircase-Spiral

## Variable Analysis

The staircase-spiral uses size in two dimensions simultaneously to encode two different quantitative variables on the same geometric element. This is a compound encoding: one step encodes both cumulative value (via width) and period increase (via height).

From my framework, this is a variable-conflict encoding: the same perceptual unit (one step) carries two distinct quantitative claims. The viewer cannot decode both claims simultaneously — they must alternate attention between width and height, or read the numeric labels.

This is distinct from the dual-axis encoding in Minard's march map (Innovation #3 from 0001): there, the geographic axis and the temporal axis are two different physical dimensions of the image, and the viewer reads them as separate layers. In Du Bois's staircase, the two variables are two dimensions of the same geometric element. The interference is more direct.

**Variable matching assessment:**
- Width (size, quantitative): appropriate for cumulative value — the eye can estimate relative widths as proportional quantities
- Height/riser (size, quantitative): appropriate for period increase — the eye can estimate relative heights as proportional quantities
- Both applied to the same element: creates conflict — the eye cannot simultaneously estimate both dimensions of a single element without deliberate attention-switching

This is not a fatal grammar error — it is a known limitation of 2D forms where multiple quantitative variables must be shown for the same objects. Scatterplots solve it by using position for two variables; the staircase-spiral uses width and height of the same element. The scatterplot solution is more legible; the staircase-spiral solution is more visually distinctive.

## Innovation Flagged

**Innovation: Same-element compound quantitative encoding** — Rubric anchor: Legibility / Visual Economy. When two quantitative variables are encoded in two size dimensions (width and height) of the same geometric element, the viewer faces a compound decoding challenge: they must estimate two independent size properties of a single object. This is more cognitively demanding than position-based compound encoding (scatterplot) or layered encoding (Minard-style separate data channels). The rubric should recognize same-element compound quantitative encoding as a specific source of cognitive load, distinct from dual-axis encoding (separate visual channels) and from categorical + quantitative dual encoding (one encoding per mark type).
