# Panel Review — Tufte on 0020: Du Bois Staircase-Spiral

## Opening

I have written that Du Bois's Paris charts are among the finest data visualizations ever produced. I maintain that assessment for the suite as a whole. For this specific chart — the staircase-spiral — my admiration is more qualified.

The staircase-spiral is a dimensional layering experiment. I respect the ambition. Minard layered five variables in his march map and succeeded because the variables mapped to natural visual properties without interference. Here, Du Bois layers three variables in a form where two of them (tread width and riser height) are both size variables applied to the same geometric object. The interference is real.

## 1. School Fidelity /20 — Score: 16

Strong advocacy grammar. The argument comes first; the form serves the argument; the data is accurate. This is the advocacy school's founding logic correctly applied.

**Score: 16/20**

## 2. Data/Story Integrity /20 — Score: 17

The property values from Georgia state records are the best available 1900 source. Du Bois's rigor is not in question. The nominal vs. real-dollar question is an undeclared tradeoff that, as Du Bois notes, goes against the argument — the real growth was larger, so the undeclared tradeoff is conservative rather than misleading.

**Score: 17/20**

---

## 3. Legibility /15 — Score: 9

The staircase-spiral imposes a form-learning tax on the viewer. This is a real cost in a public-display context. My small-multiples preference would say: six bars, one for each period, is unambiguous. The viewer does not need to learn a new encoding grammar. They extract the argument (consistent growth, culminating in a 3.5x increase) in five seconds.

The staircase-spiral requires the viewer to learn:
1. The spiral is ordered (inner = earlier, outer = later)
2. The width of each step represents total property value
3. The height of each step represents the increase from the previous period
4. The numbers confirm the visual encoding

This is four pieces of information the viewer must learn before extracting the data. For a passing visitor at a large exhibition, this is too many.

**Score: 9/15** — Form-learning tax in a public-display context is too high. Gestalt available (growth is visible) but full encoding requires instruction.

---

## 4. Visual Economy /15 — Score: 11

The dimensional layering creates a visual complexity that a bar chart would not have. The outer steps are physically much larger than the inner steps — not because the data warrants a 5x size increase, but because the spiral geometry requires it. The outer steps have more visual mass than the data merits relative to the inner steps. This is form-geometric waste (Innovation #16: center waste in radial layouts) applied to a spiral.

The form also encodes the temporal variable through position (spiral order) which is redundant with the numbering of steps. The sequential position AND the numbering both assert temporal sequence. That redundancy might be justified (the form's novelty makes redundant encoding helpful for inference), but in a simpler design it would be removed.

**Score: 11/15** — Spiral geometry creates visual mass excess in the outer steps; dual temporal encoding (position + numbering) is redundant.

## Innovation Flagged

**Innovation: Spiral geometry as quantitative distortion** — Rubric anchor: Visual Economy / Data/Story Integrity. In a spiral form, the physical size of the element representing a given period grows as the spiral expands, independent of the data. The outer steps have more visual mass (larger physical area on the page) than the inner steps, not because the data for those periods is proportionally larger, but because the spiral geometry requires larger outer elements. This is a systematic visual distortion: the visual mass of late-period elements exceeds what the data alone would warrant. Related to Innovation #16 (center waste in radial layouts) but distinct: here the distortion is in the data elements, not in the empty center. The rubric should recognize spiral geometry as introducing a systematic visual mass distortion that must be assessed relative to the data's actual magnitudes.
