# Brief — 0028 The DEGAS Rubric as a System Map

## The Subject

The DEGAS rubric is a self-improving system: works generate innovations, innovations cluster into amendments, amendments change the rubric, the changed rubric shapes future work scoring, which generates new innovations. This feedback loop has operated across 25 works and 16 clusters, producing 11 rubric versions. It is a system, and systems can be mapped.

## The Meta-Visualization Argument

This is meta-visualization — a visualization of the evaluation system itself. The subject is not external data about the world; it is the internal structure of the DEGAS project's epistemic apparatus. The system map makes visible: how the 6 rubric dimensions relate to each other, how the 16 clusters connect works to amendments, and how the feedback loop closes.

## Three Layers

- **Layer 1 (top)**: 6 rubric dimensions arranged horizontally, with connecting lines showing dimension correlations (Visual Economy — Legibility; School Fidelity — Data Integrity; Resonance — Craft)
- **Layer 2 (middle)**: 16 cluster nodes, color-coded by dimension amended, with arrows showing which clusters amended which dimensions
- **Layer 3 (bottom)**: 25 works arranged chronologically, with dots showing which clusters each work triggered

Reading direction: bottom-up. Works generate clusters; clusters amend dimensions.
