# Brief — 0034 Hurricane Noel Data Sculpture

## The Argument

Hurricane Noel struck the Caribbean in late October 2007. Miebach encoded meteorological data from the storm — wind speed, barometric pressure, wave height, temperature — into the physical structure of a woven basket. The data is embedded in the artifact: color encodes one variable, weave density another, height and pattern carry others. The encoding key is provided in adjacent documentation. The object can be read, but only if you know how to read it.

The argument is not propositional. The object argues that data has physical weight, material presence, and can inhabit a form that a viewer can hold, walk around, and experience in time rather than reading it in a single plane. The hurricane becomes a thing you can touch.

## Design Approach

Three-dimensional woven textile (basket/sculptural form). Meteorological data from Hurricane Noel (NOAA records, October 28–November 2, 2007) encoded as:
- **Color**: temperature (warm/cool gradient; red = warmer SST, blue = cooler; white = eye passage)
- **Weave density**: barometric pressure (tighter weave = lower pressure, looser = higher)
- **Height of basket wall at each radial position**: wave height at corresponding compass bearing
- **Pattern interruption**: wind speed threshold events (gusts exceeding 50 knots)

The object is roughly circular in plan, approximately 18 inches in diameter. The wall height varies from 4 to 14 inches around the circumference, mapping wave height at compass bearings. The basket is read with the encoding key as a companion document.
