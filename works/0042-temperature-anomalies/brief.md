# Brief: Animated Temperature Anomalies 1880–2024

## Subject

NASA GISS global surface temperature anomaly data, 1880–2024. Baseline: 1951–1980 average. Each year is one frame; the anomaly bar grows or shrinks from zero (baseline). Color shifts from blue (below baseline) through neutral gray to deep red (above baseline). The year is printed large in the corner. Animation speed: one year per second, 144 frames total.

## Design Intent

Rosling-style animated bar. The presenter narrates: "Watch the red grow." The innovation question: what does Execution mean for an animated work — is each frame scored individually or the animation as whole?

## School

Dynamic-statistical-graphics (Type A). This is an original design; no single canonical work holds this exact form.

## Score Target

84–89. Data-ink per frame is excellent; animation variable fully justified; color-temperature encoding intuitive but connotative.
