# Brief: Amazon Deforestation Cross-Section

## Subject

A vertical cross-section of Amazon forest at three moments: 1980, 2000, 2023. Shows how deforestation changes the vertical structure: original forest (5 layers, 40m+ canopy) to selectively logged (3 layers, gaps) to cleared cattle pasture (1 layer, grass, 1m). Environmental variables alongside: carbon stored (tonnes/hectare), biodiversity index, rainfall recycling capacity (%), soil erosion rate (tonnes/year/hectare).

## Design Intent

The gravitational hierarchy principle (U cluster, v2.2) is central: the forest's vertical structure (canopy → understory → shrub layer → herb layer → forest floor → root zone) is organized by gravity. Deforestation removes layers from the top down, collapsing the vertical structure. The visualization shows this collapse as a temporal sequence.

## School

Ecological-visualization + advocacy-visualization (Type C synthesis). The ecological school provides the grammar (vertical cross-section, species placement at correct zone boundaries, gravitational hierarchy). The advocacy school provides the argument (show the loss explicitly as a moral weight).
