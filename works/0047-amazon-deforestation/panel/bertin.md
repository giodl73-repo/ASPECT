# Bertin Review — 0047 Amazon Deforestation Cross-Section

Systematic variable assessment for the cross-section panels.

Vertical position: ecological layer (emergent canopy, canopy, understory, shrub, herb, root). Position is the most powerful spatial variable; here it encodes ecological zone. Correct and necessary — there is no other way to show vertical ecological structure except through vertical position.

Visual density: mark count per unit area within each layer. In 1980, all layers are dense with species marks. In 2023, only the 0-1m zone has marks. Density encodes ecological richness. This is texture/density as a quantitative variable — appropriate for "more species, more structure." The Cluster W (ecology-as-composition) principle applies: the honest placement of species at their correct ecological zones creates a density gradient that IS the ecological data.

Horizontal position: transect distance. The three panels share the same horizontal scale (100m transect), which makes the three states directly comparable. This is correct: same scale across comparable panels.

Position in time: panel position (left = 1980, center = 2000, right = 2023). Using panel position for time is standard in small-multiples; it requires the reader to read the year label to know which panel represents which time. This is the only encoding that requires instruction — without year labels, the panels would be hard to order.

The environmental variable bars: these are standard horizontal bar charts adjacent to each cross-section panel. Bar length encodes quantitative value; each bar is labeled with its value. The four variables (carbon, biodiversity, rainfall recycling, erosion) are correctly encoded. My concern: the bars use a different scale for each variable (carbon in t/ha, biodiversity in index units, rainfall in %, erosion in t/yr/ha). Without a common scale, the viewer cannot visually compare the rate of change across variables — they must read the numbers. This reduces retinal efficiency for the comparative question "which variable changed most dramatically?"

Recommendation: normalize all four variables to a percentage-of-1980-reference scale (1980 = 100%). Then the comparison across variables becomes visually immediate. The carbon loss (96%), biodiversity loss (92%), rainfall reduction (82%), and erosion increase (4,700%) become comparable in a single view.
