# Tufte Review — 0048 Wine Export Flow Map

I want to make a precise technical comparison between this map and the Napoleon march map, because the panel has asked whether the Napoleon map's score is earned or inflated by emotional weight.

**Technical equivalence:**
- Band width grammar: identical in both maps
- Geographic base handling: identical
- Scale declaration: identical
- Label placement at key points: identical
- Type and lettering: identical style

The Napoleon map is technically superior in one respect only: it encodes six variables (army size, direction, geography, route, temperature, time) versus the wine map's four variables (wine volume, direction, geography, route). The second register (temperature chart) in the Napoleon map is an additional dimension of information not present in the wine map.

**What this means for scoring:**
- School Fidelity: tied
- Integrity: the wine map might score slightly higher (contemporary commercial data, no uncertainty) vs. Napoleon (contemporary military data, also high quality). Essentially tied.
- Legibility: roughly comparable. The Napoleon map's two registers require more reading effort than the wine map's single register — slight edge to wine map.
- Execution: Napoleon has higher information density (more variables in the same space). This is a genuine Execution advantage for Napoleon.
- Resonance: Napoleon scores dramatically higher. Human tragedy produces experiential resonance that wine exports do not. This is not a bias — it is a real difference in what the work achieves.
- Purpose: Napoleon is advocacy/documentary (moral weight); wine is informational/commercial. The stakes are genuinely different.

**My verdict:** The wine map scores lower than Napoleon primarily on Resonance and Purpose — both legitimate scoring differences. The technical dimensions (School Fidelity, Integrity, Legibility, Execution) should produce nearly identical scores. If the wine map scores 4-5 points lower overall, the gap is explained by Resonance (2-3 points) and Purpose (2-3 points), not by any technical inferiority.

This confirms the rubric's handling: the Purpose dimension correctly captures the stakes difference between commercial data and human tragedy without inflating the Napoleon score on technical dimensions.
