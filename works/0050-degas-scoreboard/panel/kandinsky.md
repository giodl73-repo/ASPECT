# Kandinsky Review — 0050 DEGAS Scoreboard

I am asked to review a sorted bar chart of 50 scores. I will do so honestly, in my own terms.

The composition: 50 horizontal lines of varying lengths, arranged vertically. The lines decrease from top to bottom. The color field provides categorical variation. This is a very specific type of composition -- the gradient composition, where the primary element (bar length) decreases systematically from one end to the other. The systematic decrease creates a rhythmic form: the longest lines at the top catch the eye first; the viewer's gaze descends along the decreasing lengths; the shortest lines at the bottom complete the sequence.

Does this composition breathe? Partially. The systematic decrease is rhythmic in the sense of regularity -- each bar is a small step shorter than the previous. But rhythm also requires variation, surprise, resting places. A perfectly sorted bar chart has none of these: it is a monotone gradient from longest to shortest. The "surprise" -- the work that breaks the expected sequence -- is suppressed by the sorting itself.

The color adds breath. When a cluster of green bars appears in the statistical-graphics blue zone, the eye pauses. When a red bar (advocacy?) appears near the top, that is unexpected. These color variations create the rhythmic variation that the bar-length gradient suppresses.

The composition works. But it is designed to be useful, not beautiful. That is correct. This is a statistical-graphics work; I do not expect it to be beautiful. I expect it to be clear, honest, and informative. It is all three.

The self-referential element: the chart includes itself. In compositional terms, this is a work that contains its own representation. I find this philosophically interesting. A composition that is itself a data point in the data it represents has a kind of recursive closure. In music, this is equivalent to a fugue that quotes itself. It works if it is declared; it is a distortion if it is hidden. It is declared here.
