# Bertin Review — 0052 Soviet Five-Year Plan Progress Charts

The visual variable analysis of these charts is straightforward, and that is what makes this case analytically interesting. The charts are grammatically correct. Let me demonstrate.

**Visual variables in use:**

- **Position (x-axis):** Encodes category — production sector, year, plan period. Correct for ordinal/nominal data.
- **Size (bar length, y-axis):** Encodes quantity — production units, percentage achievement. Correct for ratio data. Bar length is a quantitative variable with correct perceptual properties.
- **Value (dotted vs. solid line):** Encodes target vs. achievement status. Correct binary encoding.
- **Color (hue + value):** Red fills for achievement bars encode ideological identity of the producing state. This is expressive, not quantitative — it adds meaning beyond the data. Not a grammar error; a deliberate encoding choice.
- **Shape (Soviet star iconography):** Decorative/ideological; not data-encoding. Light decorative load; does not significantly burden the reading.

The grammar is sound. Every significant visual variable is correctly matched to its data type. The encoding is efficient. Position and size carry the quantitative argument; value distinguishes plan from achievement; color and iconography carry ideological context without confusing the quantitative encoding.

The coercion is invisible to my analysis. A bar chart of false numbers looks identical to a bar chart of true numbers. The visual variable grammar cannot encode "this number was reported under lethal duress." There is no visual variable for source integrity. The charts pass the Bertin audit.

This is the analytical lesson of work 0052: the visual grammar can be fully correct while the visualization is comprehensively deceptive. Grammar correctness is not sufficient for Integrity. The rubric correctly separates these dimensions. The chart-making was competent; the data environment was coercive.

**Rubric gap noted:** The visual variable framework has no mechanism for source integrity assessment. This is not a gap in Bertin's semiology — it is an appropriate boundary: semiology analyzes the encoding, not the data provenance. The rubric needs an explicit source integrity question separate from the encoding questions.
