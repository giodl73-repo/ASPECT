# Design Description — 0053 Pharmaceutical Subgroup Forest Plot

## Visual Format

A forest plot — a standard statistical graphic in clinical trial reporting — displayed in a journal supplement or conference presentation slide. Forest plots use horizontal line segments with confidence interval error bars and a central point estimate, with a vertical reference line at the null effect (HR=1.0 or OR=1.0). Multiple rows represent different populations, subgroups, or studies.

## Layout Architecture

**Overall trial result placement:** The TRIAL-X overall result (HR 0.94, CI [0.85, 1.04]) appears as one row in the forest plot, typically at the top or bottom. It is not given visual prominence beyond its position. Its CI crosses the null line — this is visually apparent to a trained reader.

**Subgroup rows:** Eight to twelve rows showing the treatment effect in defined subgroup populations. All are presented at equal visual weight, with the exception of the >65 age group.

**Highlighted subgroup:** The >65 age group row is visually differentiated: distinct color (blue or gold highlight), bold row label, possibly an annotation "* p=0.02." Its CI [0.70, 0.97] does not cross the null line — visible significance, to a trained reader.

**Accompanying text:** Below or beside the forest plot: "Drug X demonstrated significant benefit in patients over 65 in the TRIAL-X study." The text is declarative; it does not say "in a post-hoc subgroup analysis" or "primary endpoint not significant."

**The diamond:** Many forest plots show an overall meta-analytic estimate as a diamond shape at the bottom. In this design, the diamond is omitted for the subgroup analysis presentation — correct practice, but its absence removes the visual summary of overall evidence.

## Typography and Framing

The slide or supplement is professionally produced. The font choices, color palette, and layout are consistent with peer-reviewed publication standards. The visual grammar is that of scientific communication, not commercial advertising. This is central to the coercion: the work reads as scientific reporting.

## Grammar Assessment

Forest plots are a legitimate statistical grammar. Horizontal extent of CI encodes uncertainty; point position encodes effect size; the null reference line creates the visual argument about significance. All of these are correctly applied. The visual variable matching is sound.

The coercion is entirely in:
1. The choice to foreground the subgroup analysis rather than the primary endpoint
2. The absence of explicit labeling of the subgroup finding as post-hoc
3. The accompanying text that presents the post-hoc finding as the trial's result
4. The de-emphasis of the non-significant primary endpoint through positioning and visual weight

Not a single mark is incorrect. The coercion is in the selection of which correct marks to emphasize.
