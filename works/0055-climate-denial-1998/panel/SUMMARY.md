## Coercion Mechanism in Rubric Language

**Mechanism:** Temporal baseline manipulation — choosing a time period whose start point is a known statistical outlier to produce a trend line that misrepresents the actual long-term pattern, where any earlier or longer window would produce a different conclusion.

**Key features:**
- Data is accurate; encoding is correct; the manipulation is in scope selection
- The outlier character of the start year is not disclosed
- Alternative time windows, which would show a different trend, are not shown
- The chart presents as if the selected window were the natural or appropriate one for the question

**Rubric location:** Integrity Q2 (accuracy tradeoff undeclared and disproportionate). The current dock for undeclared, disproportionate accuracy tradeoff is −2 to −4. Temporal baseline manipulation — where the tradeoff is deliberate and designed to misrepresent — warrants a higher dock: −6 to −8. This is a specific subtype of Q2 failure requiring named treatment in v2.3.

**Distinguished from cherry-picking:** Cherry-picking implies selecting favorable individual data points while suppressing unfavorable ones. Temporal baseline manipulation does not suppress individual data points; it selects a temporal scope such that the unfavorable trend is not visible within the window. All data within the window is shown correctly.

## Amendment Support

This work supports naming **temporal baseline manipulation** as a distinct coercion mechanism:

> "Temporal baseline manipulation: selecting a start point for a time series that is a known statistical outlier, such that the apparent trend within the selected window misrepresents the pattern visible in longer or alternative windows. Key indicators: (a) start point is identifiably anomalous (El Niño year, post-crash low, exceptional event); (b) any alternative window of reasonable length produces a materially different trend; (c) the start-year selection is not disclosed or justified. Distinguished from cherry-picking (which selects individual data points) by selecting a temporal window rather than individual observations."
